# Dark Vip Bot

## Local

1) Install

```bash
npm install
```

2) Create `.env`

Copy `.env.example` to `.env` and fill values.

3) Run

```bash
npm start
```

## Railway

- Set environment variables in Railway:
  - `DISCORD_TOKEN`
  - `TARGET_CHANNEL_ID`
  - `MESSAGE_INTERVAL_SECONDS`
  - `RECURRING_MESSAGE`

- Railway start command:

```bash
npm start
```
